# SinleUI

一个简洁、轻巧的现代移动应用UI框架。

### 作者有话说

这个UI框架是作者苦于Uniapp X插件市场缺失适合自己的UI库，在开发项目的过程中写的。我自己项目比较多，而且都是一些小型项目，资金不算太充裕。其他较完美的UI框架授权费用相对较高，自我感觉非常不划算，其他免费的UI框架大多有点不合项目的风格，于是自己写了一个。

由于作者是学生党，因此更新频率视情况待定。组件生态基本上是**由作者自己需求决定的**，我会自己用到什么就写什么。**若没有满足您的需求，可以自己二次开发**。

**本UI框架象征性收费1元。**当然，如果您愿意为项目贡献，也欢迎提供代码、指出Bug或打赏，这也是我创作的动力。😊

### 使用方法

1. 在uni-app插件市场右上角选择`下载插件并导入HBuilderX`，导入到对应的Uniapp X项目中即可
2. 在 App.uvue 中导入字体样式表  @import "@/uni_modules/sinle-ui/static/css/fonts.scss";
